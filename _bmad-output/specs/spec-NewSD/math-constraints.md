# Math Constraints — NewSD

> SD 域七大数学约束 + 求解器运行时机制。这些不是架构 AD(裁定性决策),而是系统动力学域的工程不变量——Wasm 内核必须实现的数值纪律。出处:PRD §2.2 + addendum §2。CAP-3/CAP-4 的可测证据依赖本文件。

## 七大数学约束

| 约束 | 工程机制 | 解决的问题 | FR / AD 锚点 |
|------|----------|------------|--------------|
| **存量断路器算法** | 拓扑排序前强行剪断所有存量流出边,区分"积分反馈环"与"代数死锁环" | 区分合法反馈与代数死锁 | FR-SIM-2, AD-5 |
| **单次求值原则** | 所有流量速率每步只算一次,查表供所有存量使用 | 消除浮点漂移致物质凭空产生/消失 | FR-SIM-3, AD-5 |
| **非负钳制机制** | 存量 < 0 时限制**流出速率**(投影到可行域使存量恰好归零),级联影响下游;隐式法下为带约束非线性求解(活动集/投影) | 水箱空了不能再流的物理现实 | FR-SIM-4, AD-8 |
| **DELAY 隐式存量** | 延迟函数不做历史队列,编译期转为串联幽灵存量 | 保证隐式求解器联立求解时的连续性 | FR-SIM-5/7, AD-6 |
| **双时间轴分离** | UI 时间轴 ≠ 仿真时间轴 | 防刚性系统数值爆炸传导到画布 | (架构级不变量) |
| **NaN 熔断保护** | Wasm 每帧收敛后做 NaN/Inf 位运算检查 + 相对量级突变检查 | 正反馈系统数值上溢时的紧急停机 | FR-SIM-6, AD-5 |
| **两遍扫描初始化** | Pass0 算初始值 → Pass1 焊死后推导流速 | 解决 t=0 时刻时序死锁 | (求解器启动纪律) |

## 求解器运行时机制(非 §2.2 约束,运行时行为)

| 机制 | 一句话定义 | FR / AD 锚点 |
|------|-----------|--------------|
| **求解器自适应降级 + 一键简化**(FR-SIM-8) | BDF 收敛监控 → 自动降级后向欧拉/显式试探步(黄点指示无感)→ 仍失败弹单按钮"简化模型以继续",用户零档位介入 | FR-SIM-8, AD-5/AD-7/AD-12 |

## 关键禁止错误简化(防止幽灵渗漏)

非负钳制必须作用于**流出速率**(投影到可行域使存量恰好归零),不得作用于**存量结果**(`if (nv<0) nv=0` 后置钳制)。

**为什么后置钳制是错的:** 被钳存量归零,但其下游流量仍按未钳速率消耗,物质凭空消失("幽灵渗漏"),与 FR-SIM-3 物质守恒承诺直接冲突。

**现状:** lovable/prototype 现实现采用后置钳制(`nv=0`),系单用户阶段捷径,架构阶段须按速率级钳制重写(FR-SIM-4 禁止错误简化条款)。

## 熔断 vs 降级边界(AD-5 / AD-15 / FR-SIM-6 / FR-SIM-8)

两类数值异常路由不同,不混淆触发:

| 异常类型 | 路由 | 用户表现 |
|---------|------|---------|
| **资源耗尽**(Wasm.Memory 64MB / 单步 wall-clock >500ms / AST MAX_NODES 5000 / 牛顿 maxIterations 100 仍不收敛) | **熔断** `[SYSTEM HALTED]` 暂停报错 | ASCII 弹窗,仿真暂停 |
| **迭代预算超限**(牛顿不收敛) | **熔断** | 同上 |
| **残差范数非收敛**(数值收敛压力) | **FR-SIM-8 降级链**(降级不阻断) | sparkline 黄点闪烁,仿真不暂停 |
| **数值域溢出**(NaN/Inf/量级突变) | **FR-SIM-6 熔断** | `[SYSTEM HALTED: NUMERICAL OVERFLOW]` |

同一牛顿不收敛不得被两 builder 分别路由到熔断(暂停)或降级(续跑)致相反结果(AD-5 Reviewer Gate hole #5)。

## FR-SIM-6 熔断阈值(MVP 默认,可调)

| 阈值 | 默认值 | 触发场景 | 调参引导 |
|------|--------|---------|---------|
| 相对初始量级上溢 | `1e6`(6 数量级) | 任一存量绝对值 > max(初始绝对值) × 1e6 | 强正反馈/指数增长模型(细菌/复利)合法跨 6 数量级,上调至 `1e8`–`1e10` |
| NaN/Inf | — | 任一存量为 NaN 或 Infinity | 不调,直接熔断 |
| 相邻步相对变化率 | `1e3` | `|value[t+dt] − value[t]| / max(|value[t]|, ε) > 1000` | 刚性瞬态/指数衰减近零阶段可能合法触发,临时上调至 `1e4`–`1e5`,瞬态结束恢复 |
| 防除零小常数 ε | `1e-12` | 分母保护 | 仅浓度趋零化学反应等合法极小绝对值场景才下调 |

阈值调参原则:避免盲目调值损伤模型洞察可靠性——若模型本应稳态却频繁接近 `1e6`,通常说明模型有发散 bug,应排查而非下调阈值。
